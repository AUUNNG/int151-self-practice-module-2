First-Class and Higher-Order Functions
ฟังก์ชั่นสามารถเก็บไว้ในตัวแปร และส่งเป็น arg หรือถูกส่งกลับจากฟังก์ชั่นอื่นได้ และความรู้พื้นฐานที่ ฟังก์ชั่นสามารถรับฟังก์ชั่นอื่นเป็นพารามิเตอร์ หรือส่งคืนฟังก์ชั่นได้ อาจารย์บอกว่ามันจะช่วยให้เราเข้าใจเรื่องต่อไป

Immutability
เน้นอธิบายขั้นตอนการทำงานของโปรแกรมอย่างละเอียด

Declarative
เน้นสิ่งที่ต้องการ อธิบายผลลัพธ์สุดท้ายที่ต้องการ โดยไม่ต้องกำหนดกระบวนการ

Pure Functions
ฟังก์ชันที่ใช้ input จาก local var เท่านั้น ข้อดีคือ จะไม่มี Side Effects โค้ดเสถียร

Impure Functions
แน่นอนว่า impure คือ ใช้ reference จาก global var ทำให้สามารถเกิด side effects ได้ เช่น รันเหมือนเดิม แต่ผลลัพธ์ไม่เหมือนเดิม เป็นเรื่องที่อาจารย์อยากให้พึงระวัง

Function Composition
ตัวอย่างในสไลด์ ก็แสดงให้เห็นว่าเราสามารถใช้ function มาเป็น พารามิเตอร์ให้กับฟังก์ชั่นอีกตัวได้

Function scope and Nested Function
เป็นไอเดียที่อาจารย์นำเสนอมา คือ เราสามารถมี ฟังก์ชันซ้อนฟังก์ชั่น สามารถเรียกฟังก์ชั่น child ได้ หรือ return ฟังก์ชัน ออกมาได้ ทำให้เข้าใจและการใช้งานขอบเขตของ ฟังก์ชั่นได้

Closures
ฟังก์ชั่นที่ยังจำ reference ได้ ลดการใช้ตัวแปร global
